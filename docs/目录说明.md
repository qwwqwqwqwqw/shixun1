## 目录详细说明

### 1. `dev_ws/src/` — ROS2 工作空间

> 负责人：B组长（guide_pkg）、D组员（aiserver_pkg）

ROS2 标准工作空间，使用 `colcon build` 编译。

#### 1.1 `guide_pkg/` — 导航核心包（B组长）

| 文件/目录 | 说明 |
|-----------|------|
| `guide_pkg/__init__.py` | Python 包初始化（必须存在） |
| `guide_pkg/guide_node.py` | **核心导航节点** — 订阅 `/command_room`（手动）和 `/face_room`（人脸），调用 Nav2 导航 |
| `guide_pkg/arrival_fusion.py` | **多传感器融合节点** — 订阅导航状态 + 门牌识别结果，发布最终到达确认 |
| `guide_pkg/utils.py` | 辅助函数：加载坐标映射、构造 PoseStamped |
| `config/classrooms.yaml` | **教室号→坐标映射**（A组员提供坐标值） |
| `config/face_room_map.yaml` | **人脸名→房间号映射**（C组员维护） |
| `srv/NavigateRoom.srv` | 自定义服务接口定义 |
| `launch/guide_launch.py` | 一键启动 guide_node + arrival_fusion |
| `package.xml` | ROS2 包依赖声明 |
| `setup.py` | Python 包安装脚本（含 entry_points） |
| `resource/guide_pkg` | ament 资源标记文件 |

#### 1.2 `aiserver_pkg/` — 指令接口包（D组员）

| 文件/目录 | 说明 |
|-----------|------|
| `aiserver_pkg/__init__.py` | Python 包初始化 |
| `aiserver_pkg/aiserver_node.py` | **TCP Socket 服务节点** — 桥接小程序与 ROS2 |
| `launch/aiserver_launch.py` | 启动文件 |
| `package.xml` | 包依赖声明 |
| `setup.py` | 安装脚本 |
| `resource/aiserver_pkg` | ament 资源标记 |

---

### 2. `vision/` — 视觉识别模块

> 负责人：C组员

#### 2.1 `doorplate/` — 门牌识别（YOLOv5 + OCR）

| 文件/目录 | 说明 |
|-----------|------|
| `doorplate_detector.py` | **门牌识别节点** — YOLO 检测门牌区域 + OCR 提取号码，发布到 `/doorplate_result` |
| `yolov5/` | YOLOv5 模型文件（源码或软链接） |
| `best.pt` | 训练好的权重文件（实际使用时替换占位） |
| `dataset/` | 训练数据集（图片+标注，**不提交 Git**） |

#### 2.2 `face/` — 人脸识别（酒店入住模拟）

| 文件/目录 | 说明 |
|-----------|------|
| `face_recognizer.py` | **人脸识别节点** — 检测人脸 → 匹配已知人脸 → 映射房间号 → 发布到 `/face_room` |
| `known_faces/` | 已知人脸照片目录（文件名与人脸映射表 key 一致） |
| `face_encodings.pkl` | 预计算人脸特征缓存（运行时生成，**不提交 Git**） |

#### 2.3 `launch/`

| 文件 | 说明 |
|------|------|
| `vision_launch.py` | 视觉模块一键启动脚本 |

# 接口定义文档 — ROS2 话题/服务

> 负责人：B组长
> 最后更新：2025-07-11

## 话题（Topics）

### 1. `/command_room` — 手动教室号指令

| 属性 | 值 |
|------|-----|
| 类型 | `std_msgs/String` |
| 发布者 | `aiserver_node`（D组员） |
| 订阅者 | `guide_node`（B组长） |
| 用途 | 小程序/Web 输入教室号，触发手动导航模式 |

**消息示例：**
```json
{ "data": "101" }
```

---

### 2. `/face_room` — 人脸识别房间号

| 属性 | 值 |
|------|-----|
| 类型 | `std_msgs/String` |
| 发布者 | `face_recognizer`（C组员） |
| 订阅者 | `guide_node`（B组长） |
| 用途 | 人脸识别成功后，发布映射的房间号，触发人脸导航模式 |

**消息示例：**
```json
{ "data": "102" }
```

---

### 3. `/navigation_status` — 导航状态反馈

| 属性 | 值 |
|------|-----|
| 类型 | `std_msgs/String` |
| 发布者 | `guide_node`（B组长） |
| 订阅者 | `aiserver_node`（D组员）、`arrival_fusion`（B组长） |
| 用途 | 实时反馈导航状态给前端和融合节点 |

**状态字符串约定：**
| 状态字符串 | 含义 |
|-----------|------|
| `"开始导航到 {room}（来源: {source}）"` | 导航已启动 |
| `"导航中..."` | 正在移动 |
| `"到达 {room}"` | 导航到达目标位置 |
| `"导航失败: {reason}"` | 导航异常 |

---

### 4. `/doorplate_result` — 门牌识别结果

| 属性 | 值 |
|------|-----|
| 类型 | `std_msgs/String` |
| 发布者 | `doorplate_detector`（C组员） |
| 订阅者 | `arrival_fusion`（B组长） |
| 用途 | 门牌识别节点输出检测到的门牌号文本 |

**消息示例：**
```json
{ "data": "101" }
```

---

### 5. `/arrival_confirmed` — 到达确认

| 属性 | 值 |
|------|-----|
| 类型 | `std_msgs/Bool` |
| 发布者 | `arrival_fusion`（B组长） |
| 订阅者 | `aiserver_node`（D组员） |
| 用途 | 融合导航到达 + 门牌匹配后的最终确认信号 |

---

## 服务（Services）

### 1. `/navigate_room` — 导航房间服务（可选扩展）

| 属性 | 值 |
|------|-----|
| 类型 | `guide_pkg/srv/NavigateRoom` |
| 服务端 | `guide_node`（B组长） |
| 服务客户端 | `aiserver_node`（D组员） |

**请求（Request）：**
```
string room_number    # 目标教室号
string source         # "manual" 或 "face"
```

**响应（Response）：**
```
bool success          # 是否成功发起导航
string message        # 描述信息
```

## 数据流图

```
小程序/Web ──WebSocket──> aiserver_node
                              │
                              ├──/command_room──> guide_node ──> Nav2 ──> 小车移动
                              │                       │
                              │                       ├──/navigation_status──> aiserver_node ──> 小程序
                              │                       └──/navigation_status──> arrival_fusion
                              │                                                    ▲
摄像头 ──> face_recognizer ──/face_room──> guide_node                              │
                                                                                │
摄像头 ──> doorplate_detector ──/doorplate_result───────────────────────────────┘
                                                                                │
                          arrival_fusion ──/arrival_confirmed──> aiserver_node ──> 小程序
```

## 坐标系

| 坐标系 | 说明 |
|--------|------|
| `map` | 全局地图坐标系（Nav2 导航目标使用） |
| `odom` | 里程计坐标系 |
| `base_link` | 机器人本体坐标系 |
| `camera_link` | 摄像头坐标系 |

---

## QoS 策略

| 话题 | QoS | 说明 |
|------|-----|------|
| `/command_room` | RELIABLE, KEEP_LAST(10) | 指令不可丢失 |
| `/face_room` | RELIABLE, KEEP_LAST(10) | 指令不可丢失 |
| `/navigation_status` | RELIABLE, KEEP_LAST(10) | 状态实时反馈 |
| `/doorplate_result` | BEST_EFFORT, KEEP_LAST(1) | 视觉结果允许丢帧 |
| `/arrival_confirmed` | RELIABLE, KEEP_LAST(1) | 确认信号不可丢失 |
